Информация о проекте

Изложен метод прогнозирования отклика биологической продуктивности на вероятные климатические изменения. Рассчитаны значения возможных изменений величин первичной биологической продукции зональной растительности для европейской территории России (ЕТР) на основе сценариев изменения климата для нового его равновесного состояния при удвоенной концентрации двуокиси углерода в атмосфере по трем моделям общей циркуляции атмосферы (CCC, UKHI, GFHI). Для рассматриваемой территории проведено сравнение рассчитанных прогнозных величин продуктивности с приростом органического вещества в растительном покрове для современных климатических условий и проанализирована тенденция изменения продуктивности зональных фитоценозов. Показано, что на территории Европейской России при изменении климата по рассматриваемым сценариям увеличится широтный диапазон районов, в которых возможны более высокие величины первичной биологической продукции по сравнению с современной. Сравнение модельных оценок изменения первичной биологической продукции по рассмотренным климатическим сценариям выявило незначительные расхождения этих значений для ЕТР. Полученные сценарные расчеты указывают на возможные тенденции динамики изменений продуктивности растительности европейской территории России при изменении климата.
